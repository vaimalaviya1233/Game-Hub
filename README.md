### :sparkles: About

**Game Hub** organizes all your games in a single, dedicated space, eliminating the need to hunt through your messy app drawer.  If a simple list isn't enough, Game Hub offers extensive customization, such as **Grid and Horizontal views**, customizable game **icons** and **names**, and detailed **game stats** to monitor your gameplay. The setup process is intuitive, fast, and non-intrusive, guiding you through every feature. Additionally, Game Hub features an integrated **in-app update system** to make sure that you are always running the latest version.

### :art: Design

The UI is crafted following the latest **Material 3 Expressive** guidelines. Game Hub works perfectly on every device, whether you use a regular phone, a foldable or a tablet.
